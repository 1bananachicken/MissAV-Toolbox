# MissAV封面下载器
### 安装
在Release中下载`missav_utils-0.1-py3-none-any.whl`文件，然后在命令行中执行以下命令：
```shell
pip install missav_utils-0.1-py3-none-any.whl
```

### 使用
基础用法
```python
from missav_utils.MissavCovers import CoverDownloader

downloader = CoverDownloader()
downloader.download()
```
